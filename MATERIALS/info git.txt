Первоначальная настройка Git
1. скачиваем и устанавливаем git
2. переходим в папку с проектом - cd C:\Users\Admin\django\craftnote
3. инициализируем свой проект - git init
3. если мы устанавливали git первоначально, то рекомендуется добавить имя пользователя и емаил:
	git config --global user.name "Iuriy Demenev"
	git config --global user.email ura7491@gmail.com
4. Добавление удаленного репозитория - git remote add origin https://github.com/ura7491/craftnote.git
5. Перемещение данных на удаленный репозиторий:
	git branch -M main
	git push -u origin main


Основные команды которыми следует пользоваться:
git status - проверка текущего стостояния коммита (коммит - контрольная точка изменения)
git add -A - подготовка для добавления файлов в коммит
git commit -m "My first project" - коммитим проект + добавляем комментарий влагом -m
git stash - архивирует изменения, для смены веток (не фиксируя данные в основной). Подробнее: https://git-scm.herokuapp.com/book/ru/v2/Инструменты-Git-Припрятывание-и-очистка
git log - отображение всех коммитов
git checkout <добавить адрес коммита> - позволяет откатить на предыдущие коммиты
